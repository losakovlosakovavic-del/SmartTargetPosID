# SmartTargetPosID

A Geometry Dash mod built with **Geode** that makes level building much faster!

## 🚀 Features
* **Instant Group Assignment:** Assign a specific group ID to any trigger directly in the editor layout.
* **One-Click Workflow:** No need to open the "Edit Group" menu for every single trigger anymore. Just click and confirm!

## 🛠️ Installation
1. Make sure you have the latest version of **Geode** installed.
2. Download the `.geode` file from the Releases page.
3. Drop it into your Geometry Dash `geode/mods` folder.
